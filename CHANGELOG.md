# Changelog

## 0.1.3 - 2026-05-15

- Исправлена ошибка 500 при создании нового пользователя через внутреннюю админ-панель.
- Новый пользователь теперь корректно получает пароль и выбранные права доступа.

## 0.1.2 - 2026-05-15

- Русифицированы формы и основные пользовательские подписи интерфейса.
- Роли, уровни доступа и подписи секретов теперь отображаются на русском языке.
- Английские названия в меню и таблицах заменены на понятные русские формулировки.

## 0.1.1 - 2026-05-15

- Добавлена страница "Версии патчей" с понятным описанием изменений.
- Во вкладке пользователя "Права доступа" юрлица и локации выбираются списком галочек.
- Исправлено сохранение пользователя: пустое поле нового пароля больше не сбрасывает старый пароль.

## 0.1.0 - 2026-05-14

- Подготовлена первая серверная линия системы оборудования.
- Добавлен production Docker compose с Gunicorn и Nginx.
- Добавлены настройки безопасности для `DJANGO_DEBUG=False`.
- В интерфейсе отображается версия приложения.
- Описан процесс выпуска патчей и серверного деплоя.
